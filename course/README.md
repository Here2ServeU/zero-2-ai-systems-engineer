# Course

This folder contains the complete week-by-week implementation course for *Zero 2 AI Systems Engineer*. Each week corresponds to one chapter of the book and one layer (or sub-layer) of the Nawex AI Reliability Platform.

## Modules

- `week-01-foundations` *(Chapters 1, 2)*
- `week-02-regulated-environments` *(Chapter 3)*
- `week-03-classical-ml-and-mlflow` *(Chapter 4)*
- `week-04-serving-models-via-apis` *(Chapter 5)*
- `week-05-containerization-with-docker` *(Chapter 6)*
- `week-06-infrastructure-as-code-terraform` *(Chapter 7)*
- `week-07-cloud-deployment` *(Chapter 8)*
- `week-08-ci-cd-for-mlops` *(Chapter 9)*
- `week-09-kubernetes-and-orchestration` *(Chapter 10)*
- `week-10-observability-and-monitoring` *(Chapter 11)*
- `week-11-aiops` *(Chapter 12)*
- `week-12-advanced-ci-cd-and-retraining-pipelines` *(Chapter 13)*
- `week-13-advanced-deployment-and-model-promotion` *(Chapter 14)*
- `week-14-observability-deep-dive-and-dashboards` *(Chapter 15)*
- `week-15-llm-customer-service` *(Chapter 16)*
- `week-16-rag-pipeline` *(Chapter 17)*
- `week-17-production-llm-systems` *(Chapter 18)*

The capstone lives at `../capstone-nawex-ai-reliability-platform/`.

Each module contains its own `README.md`, source code, lab assets, and supporting files.
